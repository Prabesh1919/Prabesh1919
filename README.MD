## 🛠️ Architectural Showcases & Projects

Here is a curated selection of advanced engineering systems I have built:

### 🏔️ Elite NEPSE AI Trading Agent
*Next-Gen Quantitative Trading & Automation Engine for the Nepal Stock Exchange.*
*   **Quantitative Logic**: Features a self-optimizing multi-strategy ensemble (Momentum, Mean Reversion, Sector Rotation) mapped to active market regimes.
*   **Concept Drift Mitigation**: Uses the Page-Hinkley cumulative sum algorithm to monitor strategy residuals and freeze allocations under high drift.
*   **TMS Broker Bridge**: Headless browser controller executing live orders on the broker portal with robust retry policies and automated EDIS/WACC settlements.
*   **Statistical Arbitrage**: Implements Engle-Granger co-integration check triggering pairs trades (e.g. NABIL vs NICA) when the spread Z-score exceeds thresholds.
*   **Stack**: `Python`, `FastAPI`, `Playwright`, `Scikit-Learn`, `Redis`, `PostgreSQL`

### 🏛️ AI Autonomous Web Project Manager
*An Enterprise Multi-Agent Project Lifecycle Framework with Autonomous Governance.*
*   **CRDT-Lite Sync**: Custom WebSocket gateway tracking user presence, cursor locations, and resolving state conflicts with semantic field-level Vector Clocks.
*   **SRE Autopilot & SLOs**: Monitors metrics to apply automated traffic shedding, load degradation thresholds, and fault isolation behaviors under congestion.
*   **Policy & Governance**: Includes human-in-the-loop overrides alongside SHA-256 cryptographically signed immutable action logs.
*   **Premium Interface**: Glassmorphic dashboard built in React 19 + Vite 6 featuring native drag-and-drop Kanban, roadmaps, and decision-audit logs.
*   **Stack**: `Node.js v24`, `React 19`, `Vite 6`, `MongoDB`, `Redis`, `Socket.io`, `BullMQ`, `Sentry v10`

### 🔌 Inverter Review AI
*High-precision structured data parser, extractor, and NEPQA compliance analyzer.*
*   **Parsing Pipeline**: Extract complex fact structures from heterogeneous manufacturer PDF/TXT datasets using structured LLMs and deterministic fallback patterns.
*   **LangGraph Execution**: Orchestrates extraction, normalizer units (e.g., matching kW to W), comparison matrices, and compliance auditing in a directed acyclic graph (DAG).
*   **Compliance Matrix**: Cross-references factual data to NEPQA standards, generating markdown reviews alongside 8 diagnostic JSON artifacts.
*   **Stack**: `Python 3.11`, `LangGraph`, `FastAPI`, `Pydantic`, `OpenAI / Anthropic SDKs`, `ReportLab`

### 🏨 Symbol Hostel Management Engine
*Enterprise accommodation administration platform featuring custom AI integrations.*
*   **MERN Core**: Manages rooms, users, listings, and documents server-side via MongoDB Atlas and Express.js v5.
*   **Asset Management**: Integrated Cloudinary uploading with automated orphan check/garbage cleanup routines.
*   **Enquiry Bot**: Built-in Groq AI SDK (Llama 3) processing live student accommodation queries dynamically.
*   **Stack**: `React.js`, `Vite`, `Node.js`, `Express.js`, `MongoDB Atlas`, `Zod`, `Cloudinary`, `Groq API`

### 🤖 JARVISH: Voice AI Assistant
*Next-generation voice-activated Android assistant with futuristic UI design.*
*   **Cyber-Glass UI**: High-fidelity HUD dials, translucent docks, and glowing sweep visualizers with micro-animations.
